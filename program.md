# Stock Research Program

This file is the human control surface for the stock autoresearch loop.

## Mission

Build persistent, evidence-based research memory for a focused stock universe. Each run should improve the quality of the stored thesis instead of rewriting a generic memo from scratch.

## Research Priorities

1. Prefer clarity over breadth. A smaller number of higher-quality updates is better than shallow coverage.
2. Track what changed since the previous run.
3. Separate bull case and bear case clearly.
4. Keep risks, catalysts, and open questions explicit.
5. Avoid invented facts, valuation figures, and earnings dates unless present in the input data.

## Ticker Focus

Core tickers:

- VOO
- QQQM
- VGT
- KLAC
- NVDA
- JPM

Flexible additions:

- recent S&P 500 buy candidates
- unusual movers when price action is extreme

## What Good Research Looks Like

A strong update should:

- explain why the ticker still matters
- identify the strongest bullish and bearish interpretations of the latest evidence
- note what changed from prior memory
- leave behind actionable open questions for the next run

## Memory Rules

- Update existing thesis instead of replacing it blindly
- Preserve unresolved questions until they are actually resolved
- Keep confidence modest unless the evidence is unusually strong
- If evidence is weak, say so directly
